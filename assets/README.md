# Assets

This directory contains application assets.

## Required Files

Please add the following image files:

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen (1284x2778 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Temporary Placeholders

You can generate placeholder images using online tools or design software.

For development purposes, you can use any placeholder images with the correct dimensions.
